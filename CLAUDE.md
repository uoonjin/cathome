# 프로젝트 개요
- **목적**: 고양이 쇼핑몰을 주제로한 포트폴리오 사이트 구현
- **주요기능**: 고양이 관련 상품을 온라인으로 구매할 수 있고 커뮤니티 역활도 함
- **타겟사용자**: 고양이용품을 소비하는 적극적인 소비자(Vocal Consumer)

# 기술 스택
- **Frontend**: HTML5, CSS3, JavaScript (Vanilla)
- **Backend/Database**: Supabase
- **배포**: Vercel
- **버전관리**: GitHub

# 코딩 규칙
## HTML
- 시맨틱 태그 우선 사용 (header, nav, main, article, section, footer)
- 주석은 한국어로 작성
- 들여쓰기는 2칸

## CSS
- 클래스명은 명확한 의미 전달
- Vanilla CSS 사용 (프레임워크 사용 금지)
- 주석으로 섹션 구분

## JavaScript
- ES6+ 문법 사용
- 가독성을 위해 함수는 작게 분리
- 주석은 한국어로 작성

# 게시판구현

## 게시판 요구사항
### 게시판이름
- 집사 커뮤니티

### 필수 기능
- 글 번호는 자동 생성
- 제목, 내용, 작성자, 작성일 표시
- 이미지 1개 업로드 가능
- 댓글 작성/삭제
- 제목 검색
- 로그인한 사람만 글쓰기
- 내가 쓴 글만 수정/삭제

### 기술 스택
**프론트엔드 (화면)**
- Vanilla JavaScript - 프레임워크 없는 순수 자바스크립트
- HTML, CSS

**백엔드 (서버/데이터베이스)**
- Supabase - 데이터베이스, 로그인, 파일 저장 기능 제공

**배포 (호스팅)**
- Vercel - 무료로 웹사이트 배포
